My Vim Config (~/.vimrc)

This is my current .vimrc. I use vim as my primary editor and it's very powerful. A lot of people at school ask why I don't use a more modern IDE like Eclipse or IntelliJ. Putting one of these poor souls in front of a headless server and asking them to exit VI is better than rand() when it comes to generating random strings.

Whenever I set up a new machine I just pull in my Vim config from Github. If you want to use it you'll get a warning if you don't have the Vim plugin, Pathogen, installed. You can ignore the warning or install Pathogen (which is really helpful in managing Vim plugins). My .vimrc is mostly suited for C style coding and uses 4 spaces for tabs. I have specific filetype configs for languages where I prefer 2 spaces like CSS and HTML. Also at the bottom are some cool shortcuts to make switching between windows easier.

" " Configuration for C-like languages.
function! SetupForCLang()
    " Highlight lines longer than 80 characters.
    au BufWinEnter * let w:m2=matchadd('ErrorMsg', '\%>80v.\+', -1)
    " Alternately, uncomment these lines to wrap at 80 characters.
    " setlocal textwidth=80
    " setlocal wrap
    "
    " " Show a colored column at 79 characters
    set colorcolumn=-1

    " Use 2 spaces for indentation.
    setlocal shiftwidth=2
    setlocal tabstop=2
    setlocal softtabstop=2
    setlocal expandtab

    " Configure auto-indentation formatting.
    setlocal cindent
    setlocal cinoptions=h1,l1,g1,t0,i4,+4,(0,w1,W4
    setlocal indentexpr=GoogleCppIndent()
    let b:undo_indent = "setl sw< ts< sts< et< tw< wrap< cin< cino< inde<"
